# HTML - CSS

Esta pasta reune meus estudos de HTML e CSS.

Aqui estou praticando os fundamentos de desenvolvimento web com exercicios e desafios do curso, evoluindo do basico para estruturas mais completas.

## O que voce encontra aqui

- Exercicios de HTML e CSS organizados por aulas.
- Desafios praticos separados por modulo.
- Projetos de treino para fixacao de conteudos como:
  - semantica HTML
  - links, imagens, midia e navegacao
  - tipografia e cores
  - seletores, pseudo-classes e modelo de caixas

## Organizacao

- `Exercicios/`: atividades de pratica de cada etapa.
- `Desafios/`: desafios por modulo do curso.

## Objetivo

Consolidar meu aprendizado em HTML e CSS por meio de pratica constante, registrando minha evolucao ao longo dos modulos.
